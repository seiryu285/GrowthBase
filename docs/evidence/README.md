# Evidence

This directory contains example payloads and public-facing evidence material for hackathon review.

Files:

- `example-offer.json` — example offer as exposed through `/offers`
- `example-artifact.json` — example artifact shape returned by the flagship paid service
- `example-receipt.json` — example receipt shape showing proof-linked delivery metadata

These files are documentation-oriented examples for repository review.
They should remain aligned with the actual API contracts and receipt/artifact schemas used by the system.
